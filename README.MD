# Desafio Funciones

Este proyecto contiene los ejercicios del desafio para la clase de funciones. TOMAR EN CUENTA QUE LOS 2 PRIMEROS EJERCICIOS SON CAMBIOS PUROS EN UN ARCHIVO DE JS, "1_FUNCION.JS Y 2_ARROW.JS" RESPECTIVAMENTE, POR LO QUE NO EXISTE ARCHIVO HTML DE ESTOS EJERCICIOS EN ESTE PROYECTO PARA SER MOSTRADO.

## Link Acceso Pages

```
https://trejosd.github.io/desafioFunciones/
```

## Instrucciones para revision de ejercicios.

- Primer ejercicio se encuentra dentro de archivo "1_funcion.js".
- Segundo ejercicio se encuentra dentro de archivo "2_arrow.js".
- Tercer ejercicio se encuentra con la union de los archivos "pintar.html, script.js".
- Cuarto ejercicio se encuentra con la union de los archivos "4_colores.html, 4_colores.js".
- Quinto ejercicio se encuentra con la union de los archivos "keydown.html, keydown.js".

## Descripción

Proyecto muestra un landing page con los links a los ejercicios 3 y 4 localizado en githubpages.
